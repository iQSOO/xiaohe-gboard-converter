# XiaoHe Gboard Converter

一个用于将中文词库转换为 Gboard 可导入格式的小鹤双拼转换工具。

> 官方博客与项目主页：
> https://www.iqsoo.com

---

## 项目介绍

本项目由 iqsoo.com 提供与维护，主要用于：

- 中文词库转换
- 小鹤双拼编码生成
- Gboard 词典导入
- 自定义输入法词库处理

适用于：

- Gboard
- 安卓输入法词库制作
- 小鹤双拼用户
- 中文输入法研究与学习

---

## 快速开始

### 安装依赖

```bash
pip install pypinyin
```

### 运行转换

```bash
python3 convert_to_xiaohe_txt.py
```

---

## 输出文件

生成文件：

```text
gboard_xiaohe.txt
```

支持直接导入到 Gboard 个人词典。
